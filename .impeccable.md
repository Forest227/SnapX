## Design Context

### Users
这是一款面向 macOS 重度效率用户的截图工具，默认先按开发者本人和小范围效率用户来设计。核心场景是快速截取屏幕局部内容，把结果立即复制到剪贴板，同时像参考卡片一样钉在桌面上，便于对照写作、设计比稿、代码实现和资料整理。

### Brand Personality
整体气质以「利落、专注、专业」为基调。它不应该像炫技型创意工具，也不应显得像企业后台，而应给人一种反应快、存在感低、但做事很稳的感觉。情绪目标是让用户觉得顺手、可信、没有负担。

### Aesthetic Direction
第一版界面采用偏原生的 macOS 工具风格，使用半透明材质、低饱和背景和清晰的高对比选区边框，强调“操作层尽量隐身，内容层保持清晰”。参考方向接近 Snipaste 与原生菜单栏工具的结合体。明确避免臃肿工具栏、过度拟物和花哨装饰。

### Design Principles
1. 一步进入截图状态，主流程优先于一切次级功能。
2. 截图交互必须直接、可预期，选区反馈要清楚。
3. 钉图窗口应该轻盈悬浮，尽量不打断当前工作流。
4. 权限、失败和退出路径都要明确，避免“点了没反应”。
5. 优先保持 macOS 原生感和系统级性能表现，而不是堆叠复杂面板。
