# SnapBoard v0.0.1

发布日期：2026-04-04

## 首次发布

SnapBoard 是一款 macOS 截图工具，设计目标是还原 Snipaste 的核心体验——截完图直接在原位进入编辑，操作流畅、无多余步骤。

**项目特点：**
- 纯 Swift + SwiftUI 实现，无第三方依赖
- 截图后原位弹出编辑浮层，不打断工作流
- 支持将截图钉在屏幕上作为参考图（浮动置顶）
- 全局快捷键可自定义，支持开机启动

## 功能

- 框选截图 — 高亮窗口单击，或拖动自由框选（⌘⇧S）
- 全屏截图 — 一键截取整个显示器（⌘⇧F）
- 截图编辑 — 截图后原位弹出编辑菜单
  - 矩形标注、高亮、文字、箭头、直线、画笔、马赛克
  - 提取文字（支持中英文 OCR）
  - 保存到下载文件夹
  - 复制到剪贴板
  - 钉住图片（浮动置顶）
- 截图历史 — 查看最近 50 张截图，支持缩放预览
- 钉图管理 — 透明度调节、鼠标穿透开关
- 全局快捷键可自定义
- 主题切换 — 跟随系统 / 浅色 / 深色
- 开机启动支持

## 系统要求

- macOS 13.0+
- 需要授权：屏幕录制、辅助功能
