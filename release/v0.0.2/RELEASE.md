# SnapBoard v0.0.2

发布日期：2026-04-05

## 新功能

### 截图历史
- 截图历史上限从 50 条提升至 100 条
- 新增截图历史快捷键（默认 ⌘⇧H），可在设置页自定义
- 截图历史按钮移至菜单栏全屏截图按钮下方，显示快捷键
- 历史列表支持多选，可批量钉住或删除
- 大图预览窗口放大至 900×650，图片自动填充并居中
- 大图预览支持滚轮直接放大缩小
- 大图预览新增"钉住"按钮

### 编辑器
- 截图后进入编辑模式，按 ESC 放弃截图（不保存历史记录）
- 按 Enter 等同点击 ✅，直接复制到剪贴板并保存历史

## Bug 修复
- 修复截图完成后历史记录未写入的问题（过渡动画路径遗漏）
